Proyecto Final

Este proyecto consiste en desarrollar un videojuego multijugador de tipo "platform fighter" para dos jugadores con conexión local. También se incluirá un editor de niveles con interfaz visual de arrastrar y soltar (drag-and-drop) que permitirá exportar mapas en formato JSON.

Multijugador Local

El juego permite conectar dos jugadores mediante hotspot, utilizando la misma dirección IP.

Editor de Mapas

Sistema de creación de niveles con interfaz drag-and-drop.

Sistema de Física

Implementa motor de física para colisiones y movimiento de personajes.

Persistencia de Datos

Los mapas creados y perfiles de usuario se almacenaran en MongoDB.

Tecnologías Utilizadas

Angular

Framework para el desarrollo de la interfaz web

  • Componente de Juego: Integra Phaser para la visualización y ejecución
  • Componente de Editor: Implementa funcionalidad drag-and-drop con Tiled
  • Exportación: Genera mapas en formato JSON
Phaser

Motor 2D para la lógica y renderizado del juego

  • Sistema de Física: Gestión de colisiones y movimiento
  • Renderizado: Procesamiento de sprites y animaciones
  • Input: Manejo de controles de jugadores
Sistema de Networking

Conexión local mediante hotspot (misma dirección IP)

  • Arquitectura: Comunicación por hilos compartidos
  • Conexión Local: Funciona sin conexión a internet
  • Sincronización: Actualización de estados en tiempo real
MongoDB

Base de datos NoSQL para almacenamiento de información

  • Mapas: Almacenamiento de niveles en formato JSON
  • Usuarios: Información de perfiles y preferencias
  • Estructura: Modelo de datos flexible
