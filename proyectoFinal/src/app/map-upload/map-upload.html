Descargar Mapa Base

Cómo crear tu mapa:

  1. Descarga e instala Tiled Map Editor
  2. Descarga el mapa base (.tmx) como punto de partida
  3. Abre el archivo .tmx en Tiled y personalízalo
  4. Exporta tu mapa como archivo .json
  5. Toma una captura de pantalla de tu mapa
  6. Sube el archivo .json y la imagen aquí
